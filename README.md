# Fish Aquarium Animation 🐠

A lightweight, interactive 2D animation project built using C++. This application simulates a dynamic fish aquarium environment featuring animated aquatic life and interactive water elements.

## 🌟 Features
* **Dynamic Fish Animation:** Multi-shaped fish moving realistically across the screen within the aquarium boundaries.
* **Interactive Water Bubbles:** Animated floating bubbles that simulate real aquarium physics and aeration.
* **Classic 2D Graphics:** Built with native graphics rendering to display distinct shapes, outlines, and smooth movement loops.

## 🛠️ Built With
* **Language:** C++
* **Graphics Library:** Native C++ Graphics (e.g., Graphics.h / BGI or standard console rendering)
